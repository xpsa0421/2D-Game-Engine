# 2D Game Engine
A 2D Game Engine developed with C++ and DirectX.  

<br />

## Key Features
* Implementation of the 2D graphics pipeline
* Texture mapping and wireframe
* 2D collision of objects
* 2D object transformations
* FPS calculation
* Input Receiving
* Sound using FMOD
* Displaying text
